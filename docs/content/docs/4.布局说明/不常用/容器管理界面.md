- 移动图形
1. 你有两种方法移动数据集
可以在“报表” 中直接暴力修改序号
![输入图片说明](https://images.gitee.com/uploads/images/2021/0702/084200_f799dec1_5500438.png "屏幕截图.png")

2. 也可以在数据集编辑器中移动，
可选择插入（带着容器一起动），替换（容器不动，就是整体布局不变化， 只移动数据和图形）
