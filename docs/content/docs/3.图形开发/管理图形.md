### 应用场景
如何快速复用已开发好的组件

### 本地管理自定义组件
如下图,点击图形编辑器中的菜单,即可管理和查看自定义图形列表
点击"加入样列"即可把当前图形加入到样列清单, 点击"取消样列"即可将所选样列移除样列清单
